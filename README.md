# To-Do List App

A simple and interactive To-Do List web application that helps users manage their daily tasks efficiently.
This project was built to practice core frontend development concepts such as DOM manipulation and dynamic UI updates using JavaScript.

## ✨ Features

* Add new tasks
* Mark tasks as completed using a checkbox
* Delete tasks from the list
* Clean and simple user interface
* Instant task updates using JavaScript

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

---

## 💡 What I Learned

* DOM manipulation using JavaScript
* Handling user inputs and events
* Updating UI dynamically without page reload
* Structuring small frontend projects

## 🔧 How to Run Locally

1. Clone the repository

```
git clone https://github.com/tejastiwari1456/To-do-list.git
```

2. Open the project folder

```
cd To-do-list
```

3. Open `to-do-list.html` in your browser.
